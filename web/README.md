### Voyant Web Server

#### Local Development

##### Dependencies

```bash
npm install
npm install -g eslint babel-eslint jscs webpack
```
##### Running Dev

```bash
NODE_ENV=development npm start
```

##### NPM Commands

```bash
# starts the app
npm start

# run lints
npm run lint

# compile frontend builds
npm run compile

# compile & watch for changes
npm run compile --watch
```
